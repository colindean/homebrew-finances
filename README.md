# Homebrew OpenCollective financial analysis

**:warning: VERY WIP as of January 2024. Don't rely on this for anything.**

This project is an analysis of Homebrew's public accounting through OpenCollective.

It is a hasty fork of [hledger_finance](https://github.com/simonmichael/hledger_finance),
the public bookkeeping of the hledger project.

## Regenerating data

Setup:

    brew bundle --no-lock
    pip3 install hledger-utils
    # until https://github.com/gooofy/drawilleplot/issues/6 is fixed
    pip3 install 'pillow>=9,<10'
    # until https://github.com/Homebrew/homebrew-core/pull/160590 is merged
    curl -o hledger-bar https://raw.githubusercontent.com/simonmichael/hledger/master/bin/hledger-bar
    sudo install hledger-bar /usr/local/bin/hledger-bar

Run:

    make clean csvs journals accounts check
    make README.md SED=sed  # in a devcontainer
    make README.md SED=gsed # using brew's gnu-sed
